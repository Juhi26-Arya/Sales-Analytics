# Sales-Analytics
It Hardware Sales Analytics Excel
In this project, I developed a sales report using Excel Pivot Tables to help businesses understand their sales performance better. This report is a powerful tool for identifying the best customer discounts, making negotiations easier, and finding new markets to explore.

The insights from these reports can help businesses improve their sales strategies, boost customer satisfaction, and increase revenue.

During this project, I learned how to:

1. Use Pivot Tables for quick data analysis.
2. Apply Power Query for smooth data processing.
3. Understand the basics of DAX (Data Analysis Expressions).
4. Use Conditional Formatting to highlight important information.
5. Make reports visually appealing and easy to read.
6. Analyze key sales metrics effectively.

Overall, this project gives businesses the insights they need to enhance their sales efforts and discover new growth opportunities through data analysis.
